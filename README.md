## 🛰️ Flagship Project

<div align="center">

| Field | Details |
|:---|:---|
| **Title** | Attitude Determination of a CubeSat using Sensor Fusion of MEMS Sensors |
| **Objective** | Real-time 3D orientation of a CubeSat using low-cost MEMS inertial sensors |
| **Algorithms** | Extended Kalman Filter · Complementary Filter · AHRS (Madgwick / Mahony) |
| **Sensors** | IMU (Accel + Gyro) · Magnetometer · Barometer |
| **Target HW** | STM32 — bare-metal C, real-time on CubeSat OBC |
| **Simulation** | MATLAB / Simulink — filter validation before hardware deployment |
| **Supervisor** | Prof. Mounir Ben Ali |
| **Institution** | CRMN — Centre de Recherche en Microelectronique et Nanotechnologie, Sousse |
| **Status** | 🔄 In progress — publishing soon on GitHub |

</div>

> **Full pipeline:** Raw IMU → Allan variance bias removal → EKF predict/update → Quaternion attitude → Euler angles
